# Oscar Nuñez

## Senior Data Engineer · Full Stack Developer · Technical Founder

10+ years building data infrastructure and distributed systems — across telecom, fintech, and AI. From ETL pipelines to cloud-native architectures, focused on reliability, scalability, and making data useful for real decisions.
